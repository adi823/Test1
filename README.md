Test App
========

## Requirements
* [NodeJS](https://nodejs.org/)
* [Bower](http://bower.io/)
* [Grunt](http://gruntjs.com/)

## Installation

Clone the reposoitory

```console
git clone https://github.com/adi823/Test1.git
```

Resolve dependencies

```console
npm install
bower install
```

Run

```console
grunt server
```
